## Mistake Categories

| Category | Meaning |
|---|---|
| A | Planless trade: no written plan or outside candidate pool |
| B | Buy point error: chased, entered early, unconfirmed breakout, weak pullback |
| C | Position sizing error: too much risk, weak-market aggression, concentration |
| D | Stop-loss error: did not execute or moved stop lower |
| E | Profit-taking error: ignored target, let large profit collapse |
| F | Logic error: thesis or market-state judgment was wrong |
| G | Normal system loss: plan followed, loss stayed inside risk budget |

Treat category G as acceptable system cost. Focus corrections on categories A-F.
